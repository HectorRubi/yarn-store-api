Documentation

Welcome to the official documentation for the Express.js CRUD API, a powerful RESTful API that allows you to interact with product, user, and category data. This documentation will guide you through the various endpoints and operations available, empowering you to perform Create, Read, Update, and Delete (CRUD) operations on the resources seamlessly.

Base URL

The base URL for all API requests is: https://yarnstoreapi.onrender.com/api/v1

Endpoints

Endpoint Method Description
/products GET Retrieve a list of all products available in the system.
/products POST Create a new product.
/products/{uuid} GET Retrieve a single product available in the system.
/products/{uuid} PATCH Update a product available in the system.
/products/{uuid} DELTE Delete a single product available in the system.
/users GET Retrieve a list of all users available in the system.
/users POST Create a new user.
/users/{uuid} GET Retrieve a single user available in the system.
/users/{uuid} PATCH Update a user available in the system.
/users/{uuid} DELTE Delete a single user available in the system.
/categories GET Retrieve a list of all categories available in the system.
/categories POST Create a new category.
/categories/{uuid} GET Retrieve a single category available in the system.
/categories/{uuid} PATCH Update a category available in the system.
/categories/{uuid} DELTE Delete a single category available in the system.

Response Format

The API returns responses in JSON format, containing relevant data or error messages. Successful requests will include a 200 OK status code, while errors will have appropriate HTTP status codes along with descriptive error messages.
